# Gender Disparities in Post-Training Employment  
### British Columbia vs Rest of Canada

This project trains machine-learning models to predict post-training employment outcomes and compares feature importance for women and men.

## Overview
- Women-only and men-only models
- Logistic Regression or Gradient Boosting
- Probability heatmaps (All Canada vs BC vs Rest of Canada)
- SHAP feature-importance visualizations
